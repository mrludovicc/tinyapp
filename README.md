## Dependencies
- Node.js
- Express
- EJS
- method-override
- bcryptjs
- cookie-session

## Getting Started
- Use `npm start` command to start the project and `npm run test` for testing.
- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
